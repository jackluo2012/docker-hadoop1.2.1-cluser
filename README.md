# docker-hadoop1.2.1-cluser
#####1. clone github repository

```
git clone https://github.com/jackluo2012/docker-hadoop1.2.1-cluser.git
```

#####2. build docker image
```
cd docker-hadoop1.2.1-cluser
sh build.sh
```
#####3. run docker image
```
sh start-container.sh
```
#####3. 打开浏览器访问
```
http://ip:50030
http://ip:50070
```
